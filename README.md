# Udemy Courses Analysis

## Project Overview

This project analyzes Udemy course data to uncover trends in online learning, course popularity, pricing strategies, and student engagement.

The analysis was performed using Python and various data visualization techniques to generate actionable insights from educational data.

---

## Objectives

- Analyze course distribution across different subjects.
- Investigate course pricing trends.
- Identify the most popular courses.
- Examine subscriber engagement patterns.
- Explore relationships between price, reviews, and subscribers.

---

## Dataset

The dataset contains information about Udemy courses, including:

- Course Title
- Subject
- Price
- Number of Subscribers
- Number of Reviews
- Published Date
- Content Duration
- Level

---

## Key Findings

- Web Development courses attracted the largest number of subscribers.
- Free courses achieved significantly higher enrollments.
- Course popularity was not always linked to higher prices.
- Business and Programming courses dominated the platform.
